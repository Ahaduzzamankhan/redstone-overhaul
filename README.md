# 🔴 Redstone Overhaul Mod

A complete redesign of Minecraft's redstone system for Fabric, featuring modern logic, AI optimization, and GPU rendering.

## 🎯 Features

### 🔌 Complete Redstone Logic Overhaul
**Deterministic signal propagation** No randomness, fully predictable. **Tick-based simulation**  Scheduled updates instead of instant propagation. **30+ component types**  Logic gates, memory cells, I/O devices. **Signal caching**  Prevents unnecessary recomputation

### 🧠 AI-Powered Circuit Optimization
 **A* pathfinding** Optimal signal routing. **Performance analysis** Identifies bottlenecks automatically. **Adaptive optimization** Self-improving circuit suggestions. **Learning mode**  Remembers successful optimizations

### 🎨 GPU-Accelerated Rendering
**Instanced rendering**  Handle thousands of components. **Animated signal flow**  Real-time visualization. **Debug overlays**  Multiple visualization modes. **Color-coded strength**  Signal intensity at a glance

### ⚡ High Performance
**Multi-threaded simulation** Parallel processing per chunk. **10,000+ components** Handle massive circuits without TPS drop. **Chunk-level parallelism** - Independent chunk processing. **Optimized data structures** - Lock-free where possible.
